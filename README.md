# Banking Application using Python

## Description
This project is a simple Banking Application developed using Python. It performs basic banking operations through a menu-driven interface. Users can check their account balance, deposit money, and withdraw money easily.

## Features
- Check account balance
- Deposit amount
- Withdraw amount
- Menu-driven console interface
- Validates available balance before withdrawal

## Technologies Used
- Python

## How It Works
The application starts with an initial balance. Users can choose from the available options in the menu:
1. Balance Check – Displays current account balance
2. Deposit Amount – Adds money to the account
3. Withdraw Amount – Withdraws money if sufficient balance is available

The program keeps updating the balance based on user transactions.

## Concepts Used
- Variables
- User Input
- Conditional Statements (`if-else`)
- Loops
- Functions
